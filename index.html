Multi-Agent Systems: Coup

Welcome to the website of our Multi-Agent Systems project. The website contains all the information regarding our project including a video of the actual implementation.

We have chosen to model the card game Coup. It is a game that in principle has simple mechanics, but can quickly get involved by a number of actions, blocks and challenges that can all be either lying, truthful or bluffing. Because of the many uncertainties that are inherent to the game, the knowledge you do have is important. As such we found the game to be a good subject for this project. We have decided to simplify the game somewhat in order to contain the number of possible worlds in the accompanying Kripke model, but for the most part the game plays as it should. This has been our motivation throughout the development: to make the game as close to the actual thing while motivating all actions of the playing agents by the epistemic logic we have been taught during this course.
